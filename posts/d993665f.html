4.2 Finite state machines

Created2023-04-16|Updated2023-07-01|Computer Science
|Post View:

A. Finite state machines

  • Finite: countable
  • Finite state machine (FSM): any device that stores its current status and whose status can change as the result of an input. Mainly used as a couceptual model for designing and describing systems. There are two main ways of representing an FSM: a state transition diagram or a state transition table.

B. State transition diagrams

A visual representation of an FSM using circles and arrows.

terms representations
circles each state
arrows transitions that occur as the result of an input
double circles the accepting or goal state
  • accepting state: the state that identifies whethrt an input has been accepted.
  • FSMs do not necessarily need to have and accepting state.

C. State transition tables

A tabular representation of an FSM showing inputs, current state and next state.

D. Finite state machines with outputs

A special type of FSMs called a MEaly machine can also produce outputs. An example of an application of a Mealy machine could be a cipher, which is an algorithm that encrypts and decrypts data, also known as code., where the input becomes transformed into another.
