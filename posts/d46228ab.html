ASCII and Unicode

ASCII is a 7-bit character set containing 128 characters. It contains the numbers from 0-9, the upper and lower case English letters from A to Z, and some special characters. The character sets used in modern computers, in HTML, and on the Internet, are all based on ASCII.

ASCII stands for American Standard Code for Information Interchange. It is a character encoding standard for electronic communication. ASCII codes represent text in computers, telecommunications equipment, and other devices. Most modern character-encoding schemes are based on ASCII, although they support many additional characters.

Unicode is a standard binary coding system that has superseded ASCII. It is a computing industry standard for the consistent encoding, representation, and handling of text expressed in most of the world’s writing systems. The standard is maintained by the Unicode Consortium, and as of March 2021, there is a total of 143,859 characters, with Unicode 14.0 covering 144 modern and historic scripts, as well as multiple symbol sets and emoji.

Error checking and correction

Parity bit is a method of checking binary codes by counting the number of 0s and 1s in the code. It is a bit added to a string of binary code that indicates whether the number of 1-bits in the string is even or odd. Parity bits are used as the simplest form of error detecting code.

Analogue and digital siganls

The little expiriment above shows how to measure the speed of sound by using a microphone and an amplifier. A microphone converts sound wave into electrical signals which is known as analogue to digital conversion (ADC) in computer science.

Microphones work by converting the changes in air pressure caused by sound waves into electrical signals. Specifically, they use a diaphragm, which is a thin, flexible membrane, to detect the changes in air pressure. When sound waves hit the diaphragm, it vibrates back and forth, creating a corresponding electrical signal that the computer can process.

Another example is MIDI(Musical Instrument Digital Interface) keyboard, like shown below:

A MIDI keyboard is a device that can send commands to a MIDI-equipped device. When you press a key or pad, it creates a MIDI message. for example, the keyboard generates a MIDI note-on message that includes information about which key was pressed, the velocity of the key press, and other control data. When the key is released, the keyboard generates a MIDI note-off message that tells the receiving device to stop playing the note. The message is sent to the device it’s connected to with the use of either a USB or MIDI cable.

Data compression

Data compression is the process of reducing the size of a file or data stream by removing redundant or irrelevant information. The purpose of data compression is to reduce the storage space required for data storage or transmission, as well as to improve the efficiency and speed of data transfer.

There are two main types of data compression: lossless and lossy compression.

  1. Lossless Compression: Lossless compression is a data compression technique that reduces the size of a file without losing any information. In lossless compression, the compressed file can be decompressed back into its original form without any loss of data. This technique is commonly used for compressing text files, executable code, and data files. Examples of lossless compression algorithms include ZIP, GZIP, and RAR.

  2. Lossy Compression: Lossy compression is a data compression technique that reduces the size of a file by removing some of the information, resulting in a compressed file that is not an exact replica of the original file. The amount of information removed depends on the compression level chosen. The higher the compression level, the more information is removed, and the more the quality of the compressed file is reduced. This technique is commonly used for compressing audio, video, and image files. Examples of lossy compression algorithms include MP3, JPEG, and MPEG.

Data compression techniques can be further classified into different categories, including dictionary-based compression, statistical compression, and run-length encoding. The choice of compression technique depends on the type of data being compressed and the desired compression ratio.

Overall, data compression is a widely used technique for reducing the size of files and data streams, enabling faster and more efficient data transfer and storage.
