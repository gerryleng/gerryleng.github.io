5.1 Number systems

Created2023-04-20|Updated2023-07-07|Computer Science
|Post View:

Programmers need to choose the correct representation of a number to produce accurate results, so it is important for them to distinguish between the different types of number.

Real number system

Type(numbers) Symbol Definition Examples
Natural N numbers made up of the digits 0 to 9 N = {0, 1, 2, 3, …}
Integer Z whole numbers Z = {…, -2, -1, 0, 1, 2, …}
Rational Q can be expressed as a fraction or quotient Q = {1.5, 2.6, -3.8, 15, …}
  • Irrational numbers: a number that cannot be represented as a fraction as it would recur
  • Real number: any positive or negative number and can contain a fractional part

Definitions of ordinal numbers

  • Ordinal number is a number used to identify position relative to other numbers. e.g. first, third, whereas
  • Cardinal number is a number that identifies the size of something. e.g. third out of 30.

Use of ordinal numbers

A one-dimensional array called fruits shows a list of names below. This particular set is well-ordered as it has an internal structure that defines the relationship between the data items. In this case, the data is made up names in ascending alphabetical order.

  • Array: A data structure where data items are grouped together under a single identifier and are then accessed based on their position.
  • Well-ordered: a special type of ordered set in which every non-empty subset has a least element. In mathematics, the well-ordering theorem, also known as Zermelo’s theorem, states that every set can be well-ordered. A set X is well-ordered by a strict total order if every non-empty subset of X has a least element under the ordering.
  • Not all ordered sets are well-ordered. For example, consider the real numbers with the usual order relation. This set is ordered because we can compare any two elements, but it is not well-ordered because there are subsets that have no least element, such as the set of all negative real numbers.

Counting and measurement

The basic use of numbers is to count and measure. We use logical reasoning to determine which types of numbers should be used in different situations to perform a particular task.

The use of natural numbers to count is common in programming. For example:

  • Indexing: Natural numbers are used to index elements in arrays and other data structures. For example, if you have an array of 10 elements, you would use natural numbers from 1 to 10 to access each element of the array.

  • Iteration: Natural numbers are often used to control loops and iteration in programming. For example, you might use a for loop to iterate through a block of code a certain number of times, with the number of iterations specified by a natural number.

  • Counting: Natural numbers are used to count the occurrence of events in programming. For example, you might count the number of times a certain condition is met, or the number of items in a list or database.

  • Bit manipulation: Natural numbers are often used in bit manipulation operations, where each bit of a number represents a binary digit (0 or 1).

  • Cryptography: Natural numbers are used extensively in cryptography, where they are used to represent keys and other cryptographic values.

The use of real numbers to measure is common in programming. For example:

  • Financial applications: Real numbers are used extensively in financial applications to represent monetary values, interest rates, and other financial metrics.

  • Scientific simulations: Real numbers are used to represent physical quantities such as distance, time, and temperature in scientific simulations.

  • Graphics programming: Real numbers are used to represent the coordinates of points, the dimensions of objects, and other properties of 3D objects in graphics programming.

  • Machine learning: Real numbers are used to represent the weights and biases of machine learning models, as well as the input and output values of the models.

  • Game development: Real numbers are used to represent the position and velocity of objects in games, as well as other properties such as health points and scores.

  • Signal processing: Real numbers are used to represent analog signals such as sound waves and radio waves in digital signal processing.

  • Robotics: Real numbers are used to represent the position and orientation of robotic arms and other robotic components.

  • GPS and navigation: Real numbers are used to represent geographic coordinates and other location data in GPS and navigation systems.
