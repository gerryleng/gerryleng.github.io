4.5 Maths for regular expressions

Created2023-04-16|Updated2023-07-01|Computer Science
|Post View:

Sets

A set is a collection of symbols on any order that do no repeat.
Common sets:

  • Natural numbers N = {0, 1, 2, 3, …}
  • Positive Integers Z⁺ = {1, 2, 3, 4, …}
  • Integers Z = {…, -2, -1, 0, 1, 2, …}
  • Rational Numbers Q = {1.5, 2.6, -3.8, 15, …}
  • Real Numbers R = {47.3, -12, π, …}
    …

Set comprehension (Set building)

Set comprehension (set building) is the process of creating sets by describing them using notation rather than listing the elements.

Mathematical symbols and their definitions:

Symbol Meaning
x ∈ S x is an member of S
∧ and
∨ or
A ⊆ B A is a subset of B
A ⊂ B A is a proper subset of B
C = A ∪ B C is the union of A and B
C = A ∩ B C is the intersection of A and B
C = A \ B C is the difference of A and B

  • ‘member’ is a description of a value or element that belongs to a set

  • ‘|’ means ‘such that’

  • ‘' can also be represented as ‘⊖’ and ‘∆’

  • subset ⊆: a set where the elements of one are entirely contained within the other; can include two sets that are exactly the same.

  • proper subset ⊂: where one set is wholly contained within another and the other set has additional elements.

  • empty set: the set that contains no values which is represented either as {} or as Ø.

Example 1
A = { 2x |x ∈ N ∧ x ≥ 1}
B = { 2x + 1 |x ∈ N ∧ x ≥ 1}
A ∩ B = Ø

Example 2
A = {0n1n | n ≥ 1}
A = {01, 0011, 000111, 00001111, …}

Finite and infinite sets

Cardinality: the number of elements in a set

  • The empty se has a cardinality of zero
    Finite set: a set where the elements can be counted using natural numbers up to a particular number. For example:

A = {1, 2, 3, 4, 5}

  • A has a cardinality of five

Infinite set: a set that is not finite. For example:

A = {1, 2, 3, 4, 5, …} or A = {x | x ∈ N ∧ x ≥ 1}

  • There is no cardinality of an infinite set

Countable set and Countably infinite sets

  • A finite set is a countable set, as it has cardinality, which means that it can be counted using natural numbers.
    Countable set: a finite set where the elements can be counted using natural numbers.

  • But not all countable sets are finite, although infinite sets do not have cardinality as we do not know the total size of the set. , for some infinite sets it is possible to go through the process of counting the elements, even though you would never reach the end.
    Countably infinite sets: sets where thr elements can be put into a one-to-one correspondence with the set of natural numbers. For example:

  • The set of all natural numbers: {1, 2, 3, 4, 5, …}

  • The set of all integers: {…, -3, -2, -1, 0, 1, 2, 3, …}

  • The set of all rational numbers: {p/q : p, q ∈ ℤ and q ≠ 0}, where ℤ denotes the set of all integers.

  • The set of all even natural numbers: {2, 4, 6, 8, …}

  • The set of all odd natural numbers: {1, 3, 5, 7, 9, …}

Each of these sets has an infinite number of elements, but they can be put into a one-to-one correspondence with the natural numbers. For example, the natural numbers can be put into a one-to-one correspondence with the even numbers by multiplying each natural number by 2. This shows that the set of even natural numbers has the same cardinality as the set of natural numbers, and is thus countably infinite.

Cartesian product

Cartesian product is combining the elements of two or more sets to create a set of ordered pairs. For example:
A = {x, y, z}
B = {1, 2, 3}

A × B would produce a set of all possible ordered pairs:

The resulting set of Cartesian product of the two sets would be: {(x,1),(x,2),(x,3),(y,1),(y,2),(y,3),(z,1),(z,2),(z,3)}

This could also be written as: A × B = {x , y | x ∈ A ∧ y ∈ B}
